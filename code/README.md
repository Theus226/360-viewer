# 360 Previsualize

A minimalist, high-performance web application for viewing 360° equirectangular images (JPG, PNG, HDR).

## Passo 1: Configuração e Dependências

To run this project locally, you need the following dependencies:

```bash
npm install three @types/three @react-three/fiber @react-three/drei lucide-react
```

## Features

- **Drag & Drop**: Drop any 360 image anywhere on the screen.
- **Performance**: Automatic texture disposal and memory management.
- **Controls**: Orbit controls with damping (inertia) and zoom.
- **Shortcuts**: Press `ESC` or `R` to close the current image.
